# home

Personal blog of Shawn Yu — Senior Industrial AI Vision Engineer.

Live: https://shawnyu916.github.io/home

## Stack

Vanilla HTML/CSS/JS, no frameworks, no build tools. GitHub Pages hosting with Cloudflare Web Analytics.

## Sections

- **Blog** — Writing on AI, industrial vision, and technology
- **Gallery** — Photography
- **Favorites** — Curated tools and resources

## Development

No setup needed. Open `index.html` directly in a browser.

## Deployment

Push to `main`. GitHub Actions auto-deploys to `gh-pages`.

## Cloudflare Analytics

Token injected at deploy time via GitHub Secret `CF_ANALYTICS_TOKEN`.
